# AAPA FOOD

AAPA FOOD is a web project showcasing authentic Kashmiri *Aanchar* and other traditional foods. The application is built using **React** and **TypeScript (TSX)**, providing a modern and responsive user experience. It highlights the rich taste and culture of Kashmiri cuisine through a simple and elegant interface.
Developed by **Muffakir Hamid**, **Shaheen Nazir**, and **Suhaib Reshi**.
